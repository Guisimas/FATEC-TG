grpconv

Descrição
Este comando usa o sistema gshadow para migrar as senhas criptografadas do /etc/group para o /etc/gshadow. Neste caso, a posição da senha em /etc/group é preenchida com um “x”.
Observações

O comando grpunconv elimina o uso do sistema gshadow de proteção de senhas.
O comando pwconv usa o sistema shadow para proteger as senhas dos usuários.
O comando pwunconv elimina o uso do sistema shadow de proteção de senhas.
