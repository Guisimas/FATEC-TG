pwconv

Descrição
Este comando usa o sistema shadow para migrar as senhas criptografadas do /etc/passwd para o /etc/shadow. Neste caso, a posição da senha em /etc/passwd é preenchida com um “x”.
Observações

O comando pwunconv elimina o uso do sistema shadow de proteção de senhas.
O comando grpconv usa o sistema gshadow para proteger as senhas dos grupos.
O comando grpunconv elimina o uso do sistema gshadow de proteção de senhas.
